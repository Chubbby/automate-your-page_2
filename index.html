Lesson 2: Using Variables and Strings in Python

What is a variable?

Variables are a value that the programmer can define with a name. An example would be Age is the name and 30 would be the value assigned. The command in Python print Age would show the value of 30. To assign a value to a variable name requires the programmer to use an equals sign '='. So Age = 30 is how this example would be coded. To change the value of Age we simply change the value on the right side of the equals sign.

How does the equal sign work?

The programmer's equal sign isn't like the mathmatical equal sign. In math both sides of the equal sign has to be the same value. However in programming the equal sign defines the value for your variable. In the case of Age = 30 the value is 30. We can redefine value of Age simply by changing 30 to 40.

How can using variables help programmers?

1.Variables allow us to store the value of data.

2.That data can be associate to a name and makes the code easier to read/follow.

3.We can manipulate a value of a variable by use of an expression. Such as age = age + 1

What is a string?

A string is a value defined between quotes and can consist of a single character to a very large number of characters. Here is simple example of a string "3". It looks like a number but Python reads it as a string because of the quotes. In Python 3 + 3 would show 6 however "3" + "3" would show "33".

Lesson 3: Programming Functions aka Procedures

What is a function or procedure?

A function also know as a procedure will take an input, perform some action to provide an output. A simple function would be sum which has number (s) for input (s) and by addition of the inputs provides the output.

Two parts of functions, making it & using it

Python uses a unique word def to start/make the code of a function. This will be followed by the function name and parameters. The parameters are in parentheses. The parameters will be replaced by values when the function is used or called. After the def line of code then you define what to do with the input parameters. The sum function mentioned before would look like this:

def sum (a):
total = a + a
return total

To use (call) a function we then write the name of the function and provide the value (s) in parentheses after it. See below:

print sum (2)
>>>4

What can functions do for programming?

Once a programmer has defined a function once then that function can be used over and over without defining it again. This avoids repetition for the programmer.

Why is the return statement needed for a function?

The return statement instructs the Python what the function should produce. Without the return statement the function will perform the action just doesn't provide an output for us to see. Luckily Python will tells us we forgot the return statement. See below:

def sum (a):
total = a + a

print sum (2)
>>>None

Lesson 4: Control Flow & While Loops

Making Decisions with Code.

The operators in Python are used to make decisions to give a boolean true or false result. Basic operands are greater than (>), equal to (==), and not equal to (!=). Notice the double equal sign is used because the single equal sign denotes assigned variable.

Decision Making using IF/Else

The decision function if is followed by an expression then below it is the block which defines the code for a true result. The block is skipped if the expression is false. The else statement allows the programmer to further define what the code can do for a false result. The else statement can contain more if functions or simply define what to return for a false result. See the example below and note the indentation is very important in programming:

def bigger (a,b):
if a > b:
r = a
else:
r = b
return r

print bigger(2,7)
7

As you can tell this if function will show the result of the larger of two numbers.

How does the OR function work?

The or function is made up of two expressions one on each side of the operand or. If the first expression is true then that result is used. If the first expression is false then the second expression is evaluted and if it is true then it is used. However if neither one is true then the result is false. See below for an example of the or function:

def is_friend(name):
return name[0] == 'D' or name[0] == 'N'

Based on the name used, the first expression would be true if the name started with a capital D and would be true result.
print is_friend('Diane')
True

However if the the name started with a capital N then the second expression would be true and that would be true result.
print is_friend('Ned')
True

If neither of them were true then the result would be false.
print is_friend('Fred')
False

Loops: Allowing code to repeat an action

In Python we use what is called a WHILE loop which starts with a test expression followed by a colon. Then below that is the block of code to run if the test expression is true. Then the code loops back to the test expression to evaluate the new variable and as long as it returns a true statement this cycle/loop will never end. Once the result is false then the loop will end. See below for example:

i = 0
while i < 10:
print i
i = i + 1

This will result in the code printing 0 to 9. Once the variable i reaches 10 the test expression will return a false result and the loop ends.

Breaking a loop early

The programmer can use a break statement to end the loop when the result is still true. This can be achieved with an if command. Once the if command result is true then you can break out of the loop. This allows the programmer to continue the code on after the loop.

Lesson 5: Structured Data - Lists & For Loops

What is a LIst?

A list is very similiar to a string but there are two differences. Strings contain characters but a list can contain anything.

Example list: p = ['O', 'U', 8, 1, 2]

These can be characters, numbers or other list. The second difference is called mutation. Mutation is changing the content of the list but doesn't change the assignment of it.

List Operations

There were two operations of list defined, append and plus. The plus operator will concantenate two list together to create a new list. The append operator adds elements to the list end but the list still has the same assignment. The append will mutalate a list while plus creates a new list.

Find Element of a List

To index a list is very similiar as the find is in a string. The function index will return the position of the first element match to the value. Example below:

p = [10, 20, 30]
print p.index(20)
1

The in function for list searches a value in the list elements. If the value is in the list it returns True otherwise False.

How does a For Loop work?

The for loop allows programmer to assign a variable to first element of list and then the block following to perform a function to that element. Once the function is done the loop starts over and the variable is assigned to next element in list, performs the function again. This continues until all elements of the list has been cycled through. Example below:

def print_all_elements(p):
for e in p:
print e

print print_all_elements([1, 2, 3])

The result would print out numbers 1, 2, and 3
